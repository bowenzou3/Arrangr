# Arrangr 

**Automatic SATB + Solo Choir Arranger**  

Arrangr is a prototype tool that converts musical input (currently MIDI or audio extracted from YouTube) into **original SATB choir arrangements with a soloist**, and generates **sheet music** (MusicXML) ready for rehearsal or performance.  

It’s designed for choir directors, a cappella groups, and anyone who wants to quickly create singable arrangements.

---

## Features

- Converts chord progressions or MIDI tracks into **SATB voicings**
- Assigns the melody to a **soloist** line
- Generates **sheet music** in MusicXML format
- Handles voice ranges to keep arrangements **singable**
- Works with **YouTube audio input** (prototype stage)
